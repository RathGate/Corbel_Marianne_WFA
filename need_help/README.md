- **ZQSD/Flèches :** Mouvement horizontal
- **Space :** Saut
- **Esc/Entrée :** Menu / Confirm / Menu cancel
- Saut contre un mur **en tenant la direction du mur** :  Walljump
- Freefall en tenant la direction du mur : Wall Slide
